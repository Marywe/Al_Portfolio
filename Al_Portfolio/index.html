PROJECTS

TIME KILLER
TAGS: Unity, Designer, Programmer, Artist
Time Killer is a 3D platformer where you have to prove your skill by completing levels in a race against time.
In each level, you have a time limit to complete it. The time you have left over in each level will accumulate in your reserve, and you will be able to use it in future levels.
COOL THINGS I DID IN THIS PROJECT:
  • Creation of the game concept.
  • Design and programming of a quick and responsive physics-based 3D controller which works with any camera angle.
  • Fast, cool, and optimized level progression.
  • Online capabilities with a registration system and a leaderboard (and the possibility of still playing offline).
  • Deal with great amount of optimizations for a target 60fps build on Web.
  • Concept, 3D modeling, texture paint, rigging and modeling.

ALPINE PALS
TAGS: Unity, Designer, Programmer
Alpine Pals is a local coop 2D platformer game in which two pals have to cooperate to get to the top of every mountain without falling.
This game was created for the Spain Game Devs Jam III. We had 7 days to create a game with the theme "union" and this was the outcome.
COOL THINGS I DID IN THIS PROJECT:
  • Design and program the whole game in just 7 days!
  • Design 5 levels with an incremental difficulty.
  • Both players are physics-based and connected by a rope (also physics-based).
  • Local coop game playable in the same keyboard/controller.
